## Модули

### `__init__.py`
Точка входа аддона. Содержит `bl_info` и функции `register()`/`unregister()`.

### `properties.py`
Определяет `ScatterSettings` — `PropertyGroup` со всеми настройками:
- `source_object` / `target_object` — указатели на объекты сцены
- `density_map` / `scale_map` / `rotation_map` — текстуры-карты
- `count`, `scale_min`, `scale_max`, `random_seed` — числовые параметры
- `align_to_normal`, `use_collection` — флаги

### `operators.py`
- `SCATTER_OT_scatter` — запускает рассеивание
- `SCATTER_OT_clear` — очищает рассеянные объекты

### `ui.py`
- `SCATTER_PT_main` — панель в боковой панели View3D (категория "Scatter")

### `scatter_core.py`
Ядро алгоритма:
1. Получает mesh целевого объекта через `evaluated_depsgraph_get`
2. Триангулирует mesh через `bmesh`
3. Выбирает грани с весами по площади
4. Для каждой точки сэмплирует карты текстур и создаёт экземпляры

### `texture_utils.py`
Утилиты для работы с текстурами:
- `load_texture_as_grayscale()` — конвертирует изображение в grayscale numpy-массив
- `sample_texture()` — билинейный сэмплинг текстуры по UV-координатам
- `get_value_from_map()` — безопасное получение значения из карты (с default)

## Алгоритм рассеивания

1. **Подготовка mesh**: триангуляция, извлечение граней и UV-развёртки
2. **Взвешенный выбор**: грани выбираются случайно с весом, пропорциональным площади (большие грани получают больше точек)
3. **Сэмплинг текстур**: для каждой точки:
   - Density: случайное число `[0,1)` сравнивается со значением из карты — если значение меньше, точка пропускается
   - Scale: значение из карты интерполируется в диапазон `[scale_min, scale_max]`
   - Rotation: значение из карты отображается в угол `[0°, 360°]`
4. **Создание экземпляров**: каждый экземпляр размещается в позиции центра грани, ориентируется по нормали, масштабируется и поворачивается
